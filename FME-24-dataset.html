FME-24 Dataset

Here are the available CSV files for the FME-24 dataset. You can download them by clicking the respective download buttons.

FME-24-Annotations.csv

CSV of emotion annotations, timings, valence/arousal changes, ISRC number and film/music metadata.

This file contains FME-24 dataset. Each row is an individual annotation - one click where one participant felt a change in emotion. The csv file contains the Start and End times of the segments where participants clicked the button - where they felt a change in emotion. It contains the Valence and Arousal values participants plotted in the valence-arousal graph, these valence and arousal numbers correspond to the segment time in the row, these are labelled 'After Arousal' and 'After Valence'. 'Before Arousal' and 'Before Valence' show the emotion time points before the change was clicked. It contains the amount of change before and after a time point - how much they differ. It contains the emotion sentence that supports the annotations done by that specific participant. It contains a familiarity rating, the participant ID and metadata from the film and music.

FME-chord-extraction-before-after.csv

CSV file of the before and after chord extraction of the timestamps in the data.

This CSV shows chord extraction using Chordino, to extract the dominant chords in the before and after sections of where participants annotated a change of emotion. The 'before' chord shows the chords present before the change and the 'after' chords show the chords after - so the AV values corresponding to the chord value is the 'after' section.

FME_24-Dataset-Librosa-Feature-Extraction.csv

CSV file of librosa features extracted from before and after timestamps of the emotion annotation in the data.

This CSV features extracted from the audio files using the librosa library. The features include: MFCC_1_Mean MFCC_2_Mean MFCC_3_Mean MFCC_4_Mean MFCC_5_Mean MFCC_6_Mean MFCC_7_Mean MFCC_8_Mean MFCC_9_Mean MFCC_10_Mean MFCC_11_Mean MFCC_12_Mean MFCC_13_Mean MFCC_1_Var MFCC_2_Var MFCC_3_Var MFCC_4_Var MFCC_5_Var MFCC_6_Var MFCC_7_Var MFCC_8_Var MFCC_9_Var MFCC_10_Var MFCC_11_Var MFCC_12_Var MFCC_13_Var Spectral_Centroid Spectral_Bandwidth Spectral_Contrast Spectral_Rolloff Zero_Crossing_Rate Chroma_1 Chroma_2 Chroma_3 Chroma_4 Chroma_5 Chroma_6 Chroma_7 Chroma_8 Chroma_9 Chroma_10 Chroma_11 Chroma_12'

FME-dataset-metadata.csv

CSV file of the metadata of the audio files in the dataset. This csv shows data on each individual audio file including the length, ISRC number, and sample timestamps.

This CSV shows metadata for each individual audio file including duration, ISRC (unique identifier), and timestamps for selected samples. Useful for locating and verifying the original track versions used.

fme-93-matched_participants.csv

CSV file of 93 participants responses from the 185.

This CSV includes the 93 matched participant data from the orignal 185 participant, this is without errors from the 185 participant data.

FME-Survey-Details.csv

CSV file of the uncleaned survey results. This CSV file contains demographics of participants, all anonymized, including blank fields where participants couldn't fill out sections.

This CSV contains full participant data including dropouts, blank fields, and optional demographic info (age, gender, musical background). This CSV shows the results per participant along with information where data couldn't be found or participants left blank/left the survey early.
